Features
이런 기능이 있어요
🔍
자동 연결 상태 진단
실행 즉시 디스플레이를 감지하고, 연결이 안 되면 원인과 해결 방법을 화면에 바로 안내합니다. 5초마다 자동 갱신.
🖱
버튼 하나로 즉시 연결
화면 복제·확장·TV만·노트북만, 4가지 모드를 클릭 또는 숫자키 1~4로 즉시 적용.
⛶
전체화면 지원
F11로 전체화면 전환. 어떤 화면 비율에서도 모든 옵션이 잘리지 않고 표시됩니다.
📋
단계별 해결 가이드
케이블 확인, TV 입력 소스 변경, 드라이버 재시작 등 단계별 해결 방법을 실시간 안내.
⚡
설치 없이 바로 실행
Python, 드라이버 등 별도 설치 불필요. ZIP 압축 해제 후 .exe 더블클릭으로 바로 실행.
🎓
교실 수업 최적화
선생님 노트북 ↔ 학급 TV 연결이 끊겼을 때를 위해 만들었습니다. 10초 안에 해결.
